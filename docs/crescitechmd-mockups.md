## Design System

### Cores
```
Primária:    #0066CC (Azul corporativo)
Secundária:  #0052A3 (Azul escuro)
Sucesso:     #10B981 (Verde)
Erro:        #EF4444 (Vermelho)
Aviso:       #F59E0B (Laranja)
Neutro 900:  #111827 (Texto)
Neutro 600:  #4B5563 (Texto secundário)
Neutro 200:  #E5E7EB (Bordas)
Neutro 50:   #F9FAFB (Background)
Branco:      #FFFFFF
```

### Tipografia
```
Família: Inter ou DM Sans
Heading 1: 48px / 600 (semibold)
Heading 2: 36px / 600
Heading 3: 24px / 600
Heading 4: 20px / 600
Body Large: 18px / 400 (regular)
Body: 16px / 400
Body Small: 14px / 400
Caption: 12px / 400
```

### Espaçamento
```
xs:  4px
sm:  8px
md:  16px
lg:  24px
xl:  32px
2xl: 48px
3xl: 64px
```
